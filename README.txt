
Cataclysm Game Launcher (c) by Northstrider

This launcher source code is an powerful software
to setup a Cataclysm game client within a few seconds

Requirements:
- Dropbox

How to setup:
- Move the files in the "Dropbox Files" folder into a
dropboc folder and publish the download links for each single file.

- Edit the sourcecode download links with the links of your
dropbox files. (direct download links!)

- Compile, start and have fun =)
